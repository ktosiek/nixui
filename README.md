NixUI
=====

NodeWebkit UI for Nix package manager


Requirement
-----------

- Nix package manager


Just Run It
-----------

To try it out, run the following command:

```
$ make just-run-it
```

and wait for the node-webkit window to appear.


Development
-----------

Build development environment

```
$ make build
```

Run UI

```
$ make develop
```
